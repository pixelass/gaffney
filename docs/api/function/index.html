    public crescentShape(options: Object): Object source 

    import {crescentShape} from '@gaffney/svg'

    Creates a cresent path for the moon svg. The shape is based on the moon phases (0 - 29). The size can be adjusted but it is recommended to keep the size at 60.

    Params:

    NameTypeAttributeDescription
    options Object

    Object of options
    options.progress number

    The progress of the crescent (default: 0 - 29 as integer)
    options.size number

    The size of the viewBox (square)

    Return:

    Object

    The value for the d attribute of an svg path and the size as properties in an object {d, size}

    public svgElement(options: Object): Object source 

    import {svgElement} from '@gaffney/svg'

    Creates an SVG as a DOM element. Requires document.

    The SVG uses two CSS variables:

    • --gaffney-svg-moon-fill-1 (default: #000)
    • --gaffney-svg-moon-fill-2 (default: #ff0)

    Define these variables at any level to change the color of the SVG.

    Params:

    NameTypeAttributeDescription
    options Object

    Object of options
    options.d string

    The d attribute value of an svg path (crescent shape to circle)
    options.size number

    The size of the viewBox

    Return:

    Object

    The svg and path elements as properties in an object {svg, path}

    public svgString(options: Object): String source 

    import {svgString} from '@gaffney/svg'

    Creates an SVG as a string. Does not require document and can be used to render SVGs server-side.

    The SVG uses two CSS variables:

    • --gaffney-svg-moon-fill-1 (default: #000)
    • --gaffney-svg-moon-fill-2 (default: #ff0)

    Define these variables at any level to change the color of the SVG.

    Params:

    NameTypeAttributeDescription
    options Object

    Object of options
    options.d string

    The d attribute value of an svg path (crescent shape to circle)
    options.size number

    The size of the viewBox

    Return:

    String

    The final SVG as a string (<svg [...]>[...]</svg>)
